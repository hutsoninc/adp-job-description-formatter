ADP Job Description Formatter

Tool used by the Hutson Inc. HR team to format job descriptions for ADP.

Instructions

  • Enter the details for each section in the text area underneath appropriate heading.
  • Any text areas left blank will not be included in the output.
  • Once all details are entered, click submit and the formatted code will be displayed in the Output section.
  • Remove any bullet points and extra spaces in list sections. Leave line breaks to separate each list item.
